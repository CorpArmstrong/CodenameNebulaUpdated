ТРЕБОВАНИЯ

### вершины модели
Все вершины модели должны находиться в пределах от -127 до +127 по X, Y и Z.


### Фейсы модели
все фейсы модели должны быть триангулированы. 
Конвертер поддерживает только треугольники.


### матерриалы и текстуры
На модели может быть какое угодно количество матерриалов.
но количество различных базовых текстур для этих матерриалов 
не должно превышать 8! А имя каждого матерриала должно начинаться 
с имени текстуры и содержать аттрибуты(свойства) этого матерриала.

таким образом каждая из восьми использующихся текстур может 
применяться неограниченное количество раз, в разных местах
и с разными атрибутами.

имена метерриалов должны строиться по вот таком принцыпу:

   texture_name.[Atr][Atr]...[Atr]

где вместо "texture_name" имя использующейся текстуры
а вместо "[Atr]" один или несколько свойтв этого матерриала
всего атрибутов семь. Их можно комбинировать.

атрибуты:
Ts - two sided            (двустороность - видим с обоих сторон)
Tr - translucent          (прозрачность - обычная прозрачность)
Md - modulated            (затемнённый и прозрачный "тонированное стекло")
Wp - weapon
Un - unlit                (матерриал на который не дествует освещение)
Fl - flat                 (обычный плоский ставиться сам по умолчанию)
En - Envmapped            (зеркальный)

например:

   windowGlassTexture.Tr

   Этот матерриал используется для стекла какого нибуть окошка
   где "windowGlassTexture" - это текстура стекла этого окошка
   А "Tr" аттрибут этого матерриала 
   (Tr - translution делает материал просвещающимся)
   
   windowFrameTexture. 

   Этот матерриал используется для рамки какого нибуть окошка 
   где windowFrameTexture - это тектура этого окошка.
   А особенных аттрибутов(свойств) этот матерриал не имеет 
   потому вконце указана просто точка "." без указания аттрибутов.

ВАЖНО
Никакие другие свойства матерриала которые вы можете наложить 
в вашем трехмерном редакторе на модели видны не будут.

Движок первого деуса способен воспроизводить только указанные 
выше атрибуты, а остальное ему "по барабану".
